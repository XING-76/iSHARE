# 聲明

專案內所有圖片、文字內容僅供個人使用 無商業用途<br>

# 介紹

MERN 專案

-   提供圖片分類、搜尋、上傳功能
-   提供留言、收藏、下載功能
-   可以使用 Google 帳號簡單創建 iSHARE 帳號

# 運用技術

-   MERN 相關技術
-   JWT
-   使用 Joi 檢驗資料
-   React Hooks
-   styled-component
-   Material-UI
-   使用 Cloudinary API 上傳圖片
